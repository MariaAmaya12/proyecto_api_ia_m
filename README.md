# Proyecto API IA - Fase 1

Este proyecto implementa una API para limpieza básica de datos usando:
- Pandas (limpieza)
- Programación Orientada a Objetos (clase `DataCleaner`)
- Pydantic (validación de Input/Output)
- FastAPI (API REST)

## Estructura del proyecto
- `limpieza/`: módulo reutilizable (pipeline + clase + schemas)
- `api/main.py`: aplicación FastAPI
